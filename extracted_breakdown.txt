🔷 1. THEORETICAL FOUNDATION (VERY IMPORTANT)
You must first define your Turing Machine formally.
📌 7-Tuple Definition
A Turing Machine is:

1. Q → Set of States
These are like “modes” of your machine.
You will need:
q0 → Start state 
q_scanRight → Move to end of input 
q_add → Perform addition 
q_subtract → Perform subtraction 
q_carry → Handle carry 
q_borrow → Handle borrow 
q_cleanup → Final formatting 
q_accept → Done successfully 
q_reject → Invalid input 
👉 Store them in code:
String currentState = "q0";

2. Σ (Input Alphabet)
Allowed input symbols:
{ 0, 1, +, - }
Example inputs:
101+11 
110-01 

3. Γ (Tape Alphabet)
Includes everything in Σ plus:
B → Blank
So:
{0, 1, +, -, B}

4. δ (Transition Function)
This is the heart of the machine.
Format:
(current_state, read_symbol) → (write_symbol, move, next_state)
Example:
(q0, 1) → (1, R, q0)
(q0, +) → (+, R, q_add)

5. Special States
q0 → start 
q_accept → success 
q_reject → error 

🔷 2. SYSTEM ARCHITECTURE (HOW YOU BUILD IT)

🧩 1. Tape (VERY IMPORTANT)
Simulate infinite tape.
👉 Use:
ArrayList<Character> tape = new ArrayList<>();
Initialize:
String input = "101+11";
for(char c : input.toCharArray()) {
    tape.add(c);
}
tape.add('B'); // blank at end

🧩 2. Head Pointer
Tracks current position.
int head = 0;
Move:
head++; // Right
head--; // Left

🧩 3. Transition Table
Store δ rules.
👉 Best way:
class Transition {
    char write;
    char move; // 'L' or 'R'
    String nextState;
}
Use HashMap:
HashMap<String, Transition> transitions = new HashMap<>();
Key format:
state + symbol
Example:
transitions.put("q0_1", new Transition('1','R',"q0"));

🧩 4. Engine (MAIN LOGIC)
Loop until halt:
while(!currentState.equals("q_accept") && !currentState.equals("q_reject")) {
    
    char read = tape.get(head);
    
    Transition t = transitions.get(currentState + "_" + read);

    tape.set(head, t.write);

    if(t.move == 'R') head++;
    else head--;

    currentState = t.nextState;
}
