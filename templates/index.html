# Machine Mode Saw Wrote Moved Switched To In Simple Words

Theory of Automata — Turing Machine Emulator for Binary Arithmetic · Built with Python Flask + HTML/CSS/JS

💡 How it Works

  • Scan — Reads your input left to right to find the operators (+/−)
  • Calculate — Adds or subtracts binary numbers pair by pair, left to right
  • Carry/Borrow — If a bit overflows, it carries over to the next position
  • Cleanup — Removes operators and writes the final answer on the tape
  • Done ✓ — Machine halts and displays the result

📐 Machine Phases

q0 Reading input
q_find_op Scanning for +/−
q_add Adding bits
q_carry Carrying over 1
q_subtract Subtracting bits
q_borrow Borrowing 1
q_cleanup Writing answer
q_accept ✓ Done!
q_reject ✗ Invalid input
